# Product

## Register

product

## Users

Security analysts, SOC operators, and sysadmins monitoring distributed infrastructure for ransomware threats. They work in high-stakes environments where every second counts. The dashboard is their glanceable command center at 2am when an alert fires.

## Product Purpose

Real-time ransomware detection dashboard using Shannon entropy analysis. Detects zero-day encryption threats without relying on signature databases. Success looks like catching an unknown threat in under 500ms and giving the operator clear visual confirmation.

## Brand Personality

Rebellious, Sharp, Intense. Underground cyberpunk energy with the precision of a military-grade instrument. Not corporate security — a guardian system with attitude. Voice is direct, technical, unapologetic.

## Anti-references

No cookie-cutter dashboards. Avoid generic admin panels, Bootstrap-style layouts, and the safe corporate blue/navy enterprise security aesthetic. No glassmorphism, no frosted glass overlays.

## Design Principles

1. **Glanceable urgency** — Every element tells the operator what matters right now. No hunting for information.
2. **The interface is the event** — Threats aren't just logged, they're felt across the entire UI. Color, motion, and layout all respond.
3. **Cinematic conviction** — The cyberpunk aesthetic isn't decoration; it reinforces the product's identity as a different kind of security tool. Commit to it fully.
4. **Data as material** — Entropy scores, hex dumps, and logs aren't text on a screen. They're the raw substance of the interface itself.
5. **No false calm** — When everything is fine, the UI is quiet but alive. When a threat is active, every pixel reacts.

## Accessibility & Inclusion

Hackathon/demo project. Aesthetic experience takes priority over strict WCAG compliance. Basic accommodations where they don't compromise the visual identity.
